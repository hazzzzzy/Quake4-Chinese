    dist/savedata/q4base/fonts/chinese/*.tga

Note: this project also ships an original TTF is not included here.
Only the derived .tga image atlases and .fontdat glyph metric tables
(rendered by src/tools/export_font.py from the original OTF at build
time) are included in the dist/ tree. To rebuild from source, place
SourceHanSansSC-Medium.otf under assets/fonts/ and run export_font.py.
